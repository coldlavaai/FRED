## Project Summary

Custom CRM system for managing clients, deals, and documents across three business sectors:
- Cryptocurrency trading
- Gold trading
- Banking instruments

**Key Features:**
- Client management with document storage
- Deal tracking for all three sectors
- Wallet address tracking and flagging
- Automated document generation (5 templates)
- Electronic signature integration
- Reporting and analytics
- Secure access with user permissions

---

## Phase 1: Core CRM System

### Fixed Price: £12,000

**What's Included:**

**1. Client Management**
- Client database with contact information
- Business details and contact person tracking
- Source/referral tracking
- Document storage (CIS, ID, verification docs)
- Client notes and activity history
- Client search and filtering

**2. Deal Management**
- Three separate deal types: Crypto, Gold, Banking
- Custom fields for each sector
- Deal status tracking (Active/Grey/Dead/Completed)
- Deal value and date tracking
- Assignment to team members
- Deal notes and updates
- Commission tracking (recording only, not payment processing)

**3. Wallet Address System**
- Store crypto wallet addresses linked to deals
- Manual flagging system (Good/Bad/Suspicious)
- Search wallet history (has this address been used before?)
- Alert system for flagged wallets
- Notes per wallet address

**4. Document Automation**
- Convert your 5 templates into auto-generating forms:
  1. Buyer booking confirmation
  2. Seller booking confirmation
  3. Terms agreement
  4. Commission agreement
  5. NDA
- Auto-populate from CRM data
- Generate PDFs
- Electronic signature integration (DocuSign or similar)
- Track signing status
- Store signed documents

**5. Reporting & Analytics**
- Filter deals by sector, status, date, team member
- View all crypto/gold/banking deals separately
- View all buyers/sellers by sector
- Export data to Excel/CSV
- Dashboard with key metrics
- Deal pipeline overview

**6. Security & User Management**
- Secure login with 2-factor authentication
- User roles and permissions (Admin vs Team Member)
- Audit log (tracks all changes)
- Encrypted storage for sensitive data
- Automatic backups

**7. Training & Support**
- User documentation
- Video training session
- 30 days post-launch support (bug fixes, questions)
